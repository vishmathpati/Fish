### Typography

- **UI font:** `[FONT_NAME]` (Geist Sans recommended — never Inter, Roboto, Poppins unless explicitly justified)

- Type scale: `text-xs (12)`, `text-sm (13)`, `text-base (14)`, `text-lg (16)`, `text-xl (18)`, `text-2xl (24)`, `text-3xl (30)`, `text-4xl (36)`

### Radius

- Base: `[0.375 | 0.5 | 0.75 | 1rem]` — stored as `--radius` CSS var
- `rounded-md` → buttons, inputs
- `rounded-lg` → cards, popovers
- `rounded-xl` → modals, dialogs
- Never use arbitrary values like `rounded-[13px]`

### Spacing

- Allowed Tailwind scale: `0, 0.5, 1, 1.5, 2, 2.5, 3, 4, 5, 6, 8, 10, 12, 16, 20, 24`
- Page shell: `max-w-7xl mx-auto px-6`
- Card padding: `p-6` default, `p-4` for compact
- Section gap: `space-y-6` or `space-y-8`

### Shadows

- `shadow-sm` → inputs, subtle cards
- `shadow-md` → popovers, dropdowns, floating menus (Notion-style `0 4px 12px rgb(0 0 0 / 0.08)`)
- `shadow-lg` → dialogs, modals

### Focus ring

- Always `ring-2 ring-primary/40 ring-offset-2` on focus-visible
- Never `outline-none` without a replacement

## Hard rules

1. All colors go through CSS variables. Zero raw hex in JSX or component files (exception: `components/ui/*` generated by shadcn, which may contain CSS var references).
2. Every interactive element has both `hover:` and `focus-visible:` states.
3. Icons: `lucide-react` only. No Heroicons, no Material Icons mixed in.
4. Forms use `react-hook-form` + `zod`. No uncontrolled inputs.
5. Radix primitives are imported only inside `src/components/ui/*`. Pages and features import from `@/components/ui/*`, never from `@radix-ui/react-*` directly.
6. Never stack two cards with identical visual treatment side by side. Differentiate by padding, elevation, or divider.
7. Empty states always have: icon/illustration + one-line headline + one-line helper + single primary CTA (if actionable).
8. Loading states use `<Skeleton>` shaped to the content, never generic spinners on primary pages.

## Component library rules

- Fundamental primitives live in `src/components/ui/` (shadcn convention)
- Composites built from primitives live in `src/components/<feature>/`
- Shared cross-feature components live in `src/components/common/`
- Never duplicate a shadcn primitive. If you need a variant, add a variant prop.

## Motion principles (when motion is applied)

- Duration: 150-300ms for most interactions. 500ms max unless specifically justified.
- Easing: `ease-out` for entrances, `ease-in` for exits, `cubic-bezier(0.4, 0, 0.2, 1)` for transforms
- Respect `prefers-reduced-motion` — disable non-essential animations for users who opt out
- No spin-for-its-own-sake, no infinite animations on primary content
